fitSharp
Fit
fitSharp includes a .NET implementation of the Framework for Integrated Test (Fit) tool, including a .NET version of FitLibrary. You can use it to test .NET applications with story tests stored in the FitNesse wiki or in plain old text files.
  • The Feature List describes what fitSharp Fit can do.
  • There are two formats for Writing Tests.
  • Testing .NET applications requires Loading Assemblies and possibly specifying App Config Files.
  • Tests can be run by Using Fitnesse or Using Html Files
